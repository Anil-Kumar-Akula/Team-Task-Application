# Team-Task-Application

# 📌 Task Management System (Jira-like)

## 📖 About the Project
This is a Jira-inspired task management application designed to streamline team collaboration between managers and employees. It allows managers to assign tasks with priorities, while employees can track and update their progress. The system includes automated notifications via Email and WhatsApp and a built-in manager review workflow.

---

## 🚀 Features

- 🧑‍💼 **Task Assignment**
  - Managers can assign tasks to team members
  - Set task priorities: High, Medium, Low

- 🔔 **Notifications**
  - Email notifications
  - WhatsApp alerts for task updates

- 🔄 **Task Workflow**
  - Pending → In Progress → Completed

- ✅ **Manager Review**
  - Managers get notified when tasks are completed
  - Approve tasks or reopen if needed

- 📊 **Dashboard**
  - Employees can view their tasks
  - Managers can monitor all team tasks

---

## 🔄 Workflow

1. Manager assigns a task  
2. Employee receives Email & WhatsApp notification  
3. Employee marks task as **In Progress**  
4. Employee completes task → marks as **Completed**  
5. Manager reviews:
   - Approves → Task completed ✅  
   - Rejects → Task reopened 🔄
